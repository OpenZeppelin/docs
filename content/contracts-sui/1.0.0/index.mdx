title: Contracts for Sui 1.0.0
---

**OpenZeppelin Contracts for Sui v1.0.0** provides two core packages:

- `openzeppelin_access` for ownership-transfer wrappers around `key + store` objects.
- `openzeppelin_math` for precise unsigned arithmetic and decimal scaling.

## Scope

This version focuses on:

- `two_step_transfer` and `delayed_transfer` access policies.
- `rounding`, `decimal_scaling`, `u8`-`u256`, and `u512` math helpers.

## Installation Reference

Add the dependencies in `Move.toml`:

```toml
[dependencies]
openzeppelin_access = { git = "https://github.com/OpenZeppelin/contracts-sui.git", subdir = "contracts/access", rev = "<release-tag>" }
openzeppelin_math = { git = "https://github.com/OpenZeppelin/contracts-sui.git", subdir = "math/core", rev = "<release-tag>" }

[addresses]
openzeppelin_access = "0x0"
openzeppelin_math = "0x0"
```

<Callout type="info">
Use the latest published release tag from `OpenZeppelin/contracts-sui` (for example `v1.0.0` when available).
</Callout>
