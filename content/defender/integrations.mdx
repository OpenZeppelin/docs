Defender seamlessly integrates with your existing tools and workflows, so you can easily secure your project throughout the secure development lifecycle.

## SDK Plugins
* [**Hardhat**, window=_blank](https://hardhat.org/): Code, deploy and upgrade your Hardhat projects through Defender.
* [**Foundry**, window=_blank](https://getfoundry.sh/): Code your Foundry projects with support of Defender.

## IDEs
* [**Remix**, window=_blank](https://remix.ethereum.org/): Deploy your Remix contracts using Defender deployment environments through the Defender Remix Plugin.

## Libraries
* [**OpenZeppelin Contracts**, window=_blank](https://www.openzeppelin.com/contracts): Speed up your smart contract development with security and performance baked in.

## Continuous Integration
* [**GitHub**](module/code): Install Code Inspector to maximize security in every project PR.
* [**Defender as Code (DaC)**](dac): Serverless Framework plugin for automated resource management.

## Key Management & Transaction Execution
* [**Safe**, window=_blank](https://app.safe.global/): Use Safe multisigs to approve Defender operations.
* [**Fireblocks**, window=_blank](https://www.fireblocks.com/): Use Fireblocks assets to approve Defender operations.
* [**Flashbots**, window=_blank](https://www.flashbots.net/): Send private transactions to prevent MEV and other attack vectors.

## Notification & Logging
* [**Datadog**, window=_blank](https://www.datadoghq.com/): Push notifications and logs to your Datadog system.
* [**PagerDuty**, window=_blank](https://www.pagerduty.com/): Configure PagerDuty to receive notifications and trigger operations.
* [**Opsgenie**, window=_blank](https://www.atlassian.com/software/opsgenie): Configure Opsgenie to receive notifications for their alert management services.
* [**Slack**, window=_blank](https://slack.com/): Configure Slack channels to receive notifications.
* [**Telegram**, window=_blank](https://telegram.org/): Configure Telegram bots to receive notifications.
* [**Discord**, window=_blank](https://discord.com/): Configure Discord channels to receive notifications.
* **Webhooks**: Configure any Webhook to manage any type of alert with complete flexibility.
* **Email**: Receive alerts via any Email.

## Threat Detection & Transaction Data
* [**Etherscan**, window=_blank](https://etherscan.io/): Visualize transactions from Defender using Etherscan-based explorers.
* [**Blockscout**, window=_blank](https://www.blockscout.com/): Visualize transactions from Defender using Blockscout-based explorers.

## Source Code
* [**Github**, window=_black](https://github.com/): Install our [Code Github application](module/code) to scan your project with every pull request to identify potential vulnerabilities and suggest improvements to enhance your code quality.

If there is any other integration you would like, please [reach out to us via the form on Defender](index#feedback)!
