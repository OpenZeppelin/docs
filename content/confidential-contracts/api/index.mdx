# API Reference

## Core Components

### [Token](token)
Smart contract token utilities and implementations for ERC7984, a confidential token standard utilizing the Zama FHE library.

- **[ERC7984](token#ERC7984)** - Reference implementation for IERC7984
- **[ERC7984ERC20Wrapper](token#ERC7984ERC20Wrapper)** - Wrapper for converting ERC20 tokens to confidential tokens
- **[ERC7984Freezable](token#ERC7984Freezable)** - Extension with freezing mechanism
- **[ERC7984ObserverAccess](token#ERC7984ObserverAccess)** - Extension allowing observers to access transfer amounts
- **[ERC7984Restricted](token#ERC7984Restricted)** - Extension implementing user account transfer restrictions
- **[ERC7984Votes](token#ERC7984Votes)** - Extension supporting confidential votes tracking and delegation
- **[ERC7984Utils](token#ERC7984Utils)** - Utility library for ERC7984 functions

### [Finance](finance)
Smart contract finance utilities and implementations for confidential financial systems.

- **[VestingWalletConfidential](finance#VestingWalletConfidential)** - Handles vesting of confidential tokens with customizable schedule
- **[VestingWalletCliffConfidential](finance#VestingWalletCliffConfidential)** - Variant with cliff period
- **[VestingWalletConfidentialFactory](finance#VestingWalletConfidentialFactory)** - Factory for batch funding of vesting wallets
- **[ERC7821WithExecutor](finance#ERC7821WithExecutor)** - Extension adding executor address for arbitrary calls

### [Governance](governance)
Smart contract governance utilities and implementations for confidential governance systems.

- **[VotesConfidential](governance#VotesConfidential)** - Abstract contract for tracking confidential voting units with delegation

### [Interfaces](interfaces)
Core interfaces for interacting with confidential contracts.

- **[IERC7984](interfaces#IERC7984)** - Interface for confidential fungible token standard
- **[IERC7984Receiver](interfaces#IERC7984Receiver)** - Interface for contracts receiving ERC7984 transfers with callbacks

### [Utils](utils)
Miscellaneous contracts and libraries with utility functions for confidential contracts.

- **[FHESafeMath](utils#FHESafeMath)** - Safe arithmetic operations for encrypted values
- **[CheckpointsConfidential](utils#CheckpointsConfidential)** - Checkpoints implementation for encrypted values
- **[HandleAccessManager](utils#HandleAccessManager)** - Contract for managing ciphertext handle access
- **[Checkpoints](utils#Checkpoints)** - Standard checkpoints for non-encrypted values
