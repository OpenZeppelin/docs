#### isKeyOrAddressZero [toc] [#Utils-isKeyOrAddressZero]
#### [!toc] [#Utils-isKeyOrAddressZero]
<APIItemCompact
  circuitSig='isKeyOrAddressZero(keyOrAddress: Either<ZswapCoinPublicKey, ContractAddress>) → Boolean'
  kind="circuit"
  stackParams="true"
  modulePrefix="Utils"
>
  Returns whether `keyOrAddress` is the zero address.

  <Callout>
    Midnight's burn address is represented as `left<ZswapCoinPublicKey, ContractAddress>(default<ZswapCoinPublicKey>)` in Compact,
    so we've chosen to represent the zero address as this structure as well.
  </Callout>
</APIItemCompact>

#### isKeyOrAddressEqual [toc] [#Utils-isKeyOrAddressEqual]
#### [!toc] [#Utils-isKeyOrAddressEqual]
<APIItemCompact
  circuitSig='isKeyOrAddressEqual(keyOrAddress: Either<ZswapCoinPublicKey, ContractAddress>, other: Either<ZswapCoinPublicKey, ContractAddress>) → Boolean'
  kind="circuit"
  stackParams="true"
  modulePrefix="Utils"
>
  Returns whether `keyOrAddress` is equal to `other`.
  Assumes that a `ZswapCoinPublicKey` and a `ContractAddress` can never be equal.
</APIItemCompact>
