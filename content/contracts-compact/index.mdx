**A library for secure smart contract development** written in Compact for [Midnight](https://midnight.network/).
This library consists of modules to build custom smart contracts.

<Callout type='warn'>
This repo contains highly experimental code. Expect rapid iteration. **Use at your own risk.**
</Callout>

## Installation

Make sure you have [nvm](https://github.com/nvm-sh/nvm) and [yarn](https://yarnpkg.com/getting-started/install) installed on your machine.

Follow Midnight’s [compact installation](https://docs.midnight.network/develop/tutorial/building/#midnight-compact-compiler) guide and confirm that `compactc` is in the `PATH` env variable.

```bash
$ compactc

Compactc version: 0.24.0
Usage: compactc.bin <flag> ... <source-pathname> <target-directory-pathname>
       --help displays detailed usage information
```

### Project setup

Clone the repository:

```bash
git clone git@github.com:OpenZeppelin/midnight-contracts.git
```

`cd` into it and then install dependencies and prepare the environment:

```bash
nvm install && \
yarn && \
yarn prepare
```

Speed up the development process by skipping the prover and verifier key file generation:

```bash

```
