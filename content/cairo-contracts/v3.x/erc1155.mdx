ERC1155 is a novel token standard that aims to take the best from previous standards to create a [**fungibility-agnostic**](tokens.mdx#different-kinds-of-tokens) and **gas-efficient** [token contract](tokens.mdx#but_first_coffee_a_primer_on_token_contracts).

## Extensions

Additionally, there are multiple custom extensions, including:

* [ERC-1155 Burnable](erc1155-burnable.mdx): Optional Burnable extension of the ERC-1155 standard.
* [ERC-1155 Metadata URI](erc1155-metadata-uri.mdx): Optional extension that adds a token metadata URI.
* [ERC-1155 Pausable](erc1155-pausable.mdx): A primitive to pause contract operation like token transfers, minting and burning.
* [ERC-1155 URI Storage](erc1155-uri-storage.mdx): A more flexible but more expensive way of storing URI metadata.
* [ERC-1155 Supply](erc1155-supply.mdx): Extension of the ERC-1155 standard that adds tracking of total supply per token id.
