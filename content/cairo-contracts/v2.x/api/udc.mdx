#### `deploy_contract(ref self: ContractState, address: ContractAddress, deployer: ContractAddress, not_from_zero: bool, class_hash: ClassHash, calldata: Span<felt252>, salt: felt252) -> ContractAddress` *external*

Deploys a contract through the Universal Deployer Contract.

When `not_from_zero` is `true`, `salt` is hashed with the caller address and the modified salt is passed to the inner `deploy_syscall`. This type of deployment is [origin-dependent](/udc.adoc#origin_dependent).

When `not_from_zero` is `false`, the deployment type is [origin-independent](/udc.adoc#origin_independent).

Emits a [ContractDeployed](#contractdeployed-event) event.
